# Architecture Decisions

Each file in this directory records one significant decision:
- What we chose
- Why we chose it over the alternatives
- What trade-offs we accepted
- When we should revisit this decision

**Template:**
```markdown
# Why [X] over [Y]
**Date:** [date]
**Context:** [what problem we were solving]
**Decision:** [what we chose]
**Rationale:** [why this, not that]
**Trade-offs:** [what we gave up]
**Revisit if:** [conditions that would make us reconsider]
```
